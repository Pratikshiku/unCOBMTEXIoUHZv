## Abo Library Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/054a50bb08b342f880b0fa1f6ea8e154/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Book Information, Favorites, Borrowing, Comments
5. Announcement Information
6. Message Information
7. Personal Center
8. My Favorites
9. Change Password
10. Modify Personal Information
11. My Book Borrowing
12. My Book Return
13. Pay Fines
14. My Favorites


**Administrator Functions:**
1. Login
2. Change Password
3. User Management
4. Book Category Management
5. Book Information Management
6. Book Borrowing Management
7. Book Return Management
8. Fine Payment Management
9. Message Board Management
10. Carousel Management
11. Announcement Information
